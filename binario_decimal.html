¿Qué es el Sistema Binario?

El sistema binario es un sistema de numeración que utiliza solo dos símbolos: 0 y 1. Es la base de toda la computación moderna porque los circuitos electrónicos pueden representar fácilmente estos dos estados (apagado/encendido o bajo/alto voltaje). Cada dígito binario se llama "bit". A diferencia del sistema decimal (base 10), donde cada posición es una potencia de 10 (1, 10, 100...), en binario (base 2), cada posición es una potencia de 2 (1, 2, 4, 8, 16...). Para encontrar el valor decimal, simplemente suma los valores posicionales de todos los bits que están en '1'.

¿Cómo funciona?

Este método se basa en la probabilidad. "Lanzamos" miles de puntos aleatorios dentro de un cuadrado que contiene un cuarto de círculo.
La relación entre el área del cuarto de círculo y el área del cuadrado es Área Círculo / Área Cuadrado = (πr² / 4) / r² = π / 4.
Por lo tanto, podemos esperar que la proporción de puntos que caen dentro del círculo sea aproximadamente igual a π / 4. Despejando π, obtenemos la fórmula:
π ≈ 4 * (puntos dentro / puntos totales)
